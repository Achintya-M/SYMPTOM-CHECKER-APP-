# 🩺 HealthGuard - AI Powered Symptom Checker

## Overview
HealthGuard is a MERN Stack healthcare application that provides symptom analysis, personalized diet plans, yoga recommendations, health monitoring and secure authentication.

## Features
- Secure Login & Registration (JWT)
- Symptom Checker
- Personalized Diet Plan
- Exercise & Yoga Recommendation
- Health Monitoring Dashboard
- MongoDB Database
- Responsive UI

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt
